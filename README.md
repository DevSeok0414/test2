### 1. 개발 목적

- 학원에서 진행한 JSP MVC 프로젝트로는 부족하다고 느껴서 Spring FrameWork를 개인적으로 더 공부하여 MVC Project를 진행함.
- 게시판에 여러 기능을 추가해서 보다 깊이 있게 Spring MVC를 학습해보고자 함
